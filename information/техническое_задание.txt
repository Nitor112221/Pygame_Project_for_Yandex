При запуске меню, с кнопками: play, control, language, exit
Меню:
	exit - закрывает игру
	language - открывает доп окошко по смене языка
	control - открывает доп окошко для просмотро кнопок для управления
	должна быть система локализации с удобным добавлением новых языков
	также должна быть кнопка выключения/выключения музыки
	play - октрывает карту игрового мира
	map editor - октрывает редактор карт (только для разработчиков)
Карта игрового мира:
	открываеться большая карта, на которой должны будут располагаться точки, которые будут браться из файла, благадаря чему можно делать и сохранять прогресс
	перемещение по карте с помощью стрелочек
	при наведении на метку он подпрыгивает, и при нажатии на неё она запускает свой уровень
Игра:
	Элементы: тайлы, игрок, враги
		Тайлы:
			удобная система добавления новых тайлови
			возможность делать тайлы не осязаемыми
        Игрок:
            должен перемещаться на кнопки из файла options.txt
            должен уметь прыгать на кнопку из файла options.txt
            на левый клик мыши должен атаковать
        Враги:
            если заметили игрока, должны идти к нему
            враги не должны видеть через стены
            если игрок близко, они должны атаковать его
    Должна быть система оружий и нанесения урона, что бы можно было легко добавлять новое
    Должна быть быстрая система добавления новых врагов
    При смерти должно появляться окно поражения с 3 кнопками (меню, карта, повторить)
    При нажатии на Esc должна открываться меню паузы, а игра останавливаться
    При прохождении игрока за правую границу экрана, уровень должен считаться пройденым и выходить на карту
Редкатор карт:
    Поле, для редактирования
    Внизу список тайлов, который можно прокручивать
    При нажатии на стрелочки, поле (или список тайлов) должены двигаться
    При колёсике мыши поле должно маштабироваться
    При левом клике выбранных блок ставится, правом убирается
    При нажатии ctrl + z последний поставленный блок убирается
